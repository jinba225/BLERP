# 更新日志

本文档记录了 BetterLaser ERP 系统的所有重要变更。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
并且本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [未发布]

### 新增
- 完整的ERP系统架构设计
- 用户管理和权限控制系统
- 客户关系管理(CRM)模块
- 供应商管理模块
- 产品信息管理(PIM)模块
- 库存管理系统
- 销售订单管理
- 采购订单管理

- 报表中心

### 技术特性
- 基于 Django 4.2 的后端架构
- Tailwind CSS 响应式前端界面
- RESTful API 设计
- MySQL 数据库支持
- Redis 缓存系统
- Celery 异步任务处理
- Docker 容器化部署
- Nginx 反向代理配置

### 安全特性
- JWT 身份认证
- 基于角色的访问控制(RBAC)
- 操作审计日志
- 数据加密存储
- SQL 注入防护
- XSS 攻击防护

## 版本说明

### 版本号格式
- 主版本号：不兼容的 API 修改
- 次版本号：向下兼容的功能性新增
- 修订号：向下兼容的问题修正

### 发布周期
- 主版本：每年1-2次
- 次版本：每季度1次
- 修订版本：根据需要随时发布

### 支持政策
- 当前版本：完全支持
- 前一个主版本：安全更新支持
- 更早版本：不再支持

详细的升级指南请参考 `docs/upgrade.md` 文档。
