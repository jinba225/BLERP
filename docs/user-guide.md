## 系统概述

BetterLaser ERP 是一套专为激光设备制造企业设计的企业资源规划系统，集成了客户关系管理、供应链管理、生产管理、财务管理等核心业务功能。

### 主要特性

- **模块化设计**: 各功能模块相对独立，便于使用和维护
- **实时数据**: 所有数据实时同步，确保信息准确性
- **权限控制**: 基于角色的权限管理，保障数据安全
- **移动友好**: 响应式设计，支持移动设备访问
- **可扩展性**: 支持自定义字段和业务流程

## 登录系统

### 访问系统

1. 打开浏览器，输入系统地址
2. 进入登录页面

### 登录步骤

1. **输入用户名**: 输入管理员分配的用户名
2. **输入密码**: 输入对应的密码
3. **记住我**: 可选择记住登录状态
4. **点击登录**: 完成登录过程

### 忘记密码

如果忘记密码，请联系系统管理员重置密码。

## 仪表盘

登录成功后，您将看到系统仪表盘，这里显示了业务概况和快速操作入口。

### 主要内容

- **业务统计卡片**: 显示销售额、订单数、库存预警等关键指标
- **销售趋势图**: 展示最近的销售趋势
- **最近订单**: 显示最新的订单信息
- **快速操作**: 提供常用功能的快速入口
- **系统状态**: 显示系统运行状态

### 快速操作

- 新建订单
- 创建报价
- 采购申请
- 入库单
- 出库单
- 查看报表

## 客户管理

客户管理模块帮助您维护客户信息，管理客户关系。

### 客户列表

1. **查看客户**: 在左侧菜单点击"客户管理"
2. **搜索客户**: 使用搜索框快速查找客户
3. **筛选客户**: 按类型、等级、状态筛选
4. **排序**: 点击列标题进行排序

### 新增客户

1. 点击"新增客户"按钮
2. 填写客户基本信息：
   - 客户编码（系统自动生成或手动输入）
   - 客户名称
   - 客户类型（个人/企业/政府/其他）
   - 客户等级（A/B/C/D级）
   - 联系信息
   - 地址信息
   - 财务信息
3. 点击"保存"完成创建

### 编辑客户

1. 在客户列表中找到要编辑的客户
2. 点击"编辑"按钮
3. 修改相关信息
4. 点击"保存"确认修改

### 客户详情

点击客户名称可查看详细信息：
- 基本信息
- 联系人列表
- 地址信息
- 销售历史
- 应收账款
- 合同记录

### 联系人管理

每个客户可以添加多个联系人：
1. 进入客户详情页
2. 点击"联系人"标签
3. 点击"添加联系人"
4. 填写联系人信息
5. 设置联系人类型和权限

## 供应商管理

供应商管理功能与客户管理类似，用于维护供应商信息。

### 主要功能

- 供应商基本信息管理
- 联系人管理
- 产品目录管理
- 供应商评估
- 采购历史查询

### 供应商评估

系统支持对供应商进行定期评估：
1. 进入供应商详情页
2. 点击"评估记录"
3. 点击"新增评估"
4. 填写评估内容：
   - 质量得分
   - 交货得分
   - 服务得分
   - 价格得分
   - 综合评价
5. 保存评估结果

## 产品管理

产品管理是系统的核心模块，管理所有产品信息。

### 产品分类

1. **查看分类**: 产品按层级分类管理
2. **新增分类**: 点击"新增分类"创建新的产品分类
3. **编辑分类**: 修改分类信息
4. **删除分类**: 删除不需要的分类（需确保分类下无产品）

### 产品信息

#### 基本信息
- 产品编码
- 产品名称
- 产品分类
- 品牌
- 产品类型（原材料/半成品/成品/服务）

#### 规格信息
- 产品描述
- 技术规格
- 型号
- 计量单位

#### 价格信息
- 成本价
- 采购价
- 销售价

#### 库存设置
- 最小库存
- 最大库存
- 再订货点

### BOM管理

BOM（物料清单）定义了产品的组成结构：

1. **创建BOM**: 
   - 选择产品
   - 添加组成物料
   - 设置用量和单位
   - 定义损耗率

2. **BOM版本管理**:
   - 支持多版本BOM
   - 版本生效和失效管理
   - 版本比较功能

## 库存管理

库存管理模块提供实时的库存信息和库存操作功能。

### 库存查询

1. **实时库存**: 查看所有产品的当前库存
2. **库存明细**: 按仓库、库位查看详细库存
3. **库存预警**: 显示低于最小库存的产品
4. **库存分析**: 库存周转率、呆滞库存分析

### 库存操作

#### 入库操作
1. 点击"入库单"
2. 选择入库类型（采购入库/生产入库/其他入库）
3. 填写入库信息
4. 添加入库明细
5. 确认入库

#### 出库操作
1. 点击"出库单"
2. 选择出库类型（销售出库/生产领料/其他出库）
3. 填写出库信息
4. 添加出库明细
5. 确认出库

#### 库存调拨
1. 点击"库存调拨"
2. 选择源仓库和目标仓库
3. 添加调拨产品和数量
4. 提交调拨申请
5. 审核通过后执行调拨

#### 库存盘点
1. 创建盘点计划
2. 分配盘点人员
3. 执行盘点操作
4. 录入盘点结果
5. 生成盘点差异报告
6. 处理盘点差异

### 仓库管理

- 仓库基本信息设置
- 库位管理
- 仓库容量管理
- 仓库权限设置

## 销售管理

销售管理模块涵盖从报价到收款的完整销售流程。

### 销售报价

1. **创建报价单**:
   - 选择客户
   - 添加报价产品
   - 设置价格和数量
   - 填写交货期
   - 添加条款说明

2. **报价审核**:
   - 提交审核
   - 审核通过/拒绝
   - 修改报价

3. **报价转订单**:
   - 客户确认后转为正式订单

### 销售订单

1. **创建订单**:
   - 从报价单转换或直接创建
   - 确认产品、数量、价格
   - 设置交货期
   - 选择付款方式

2. **订单跟踪**:
   - 订单状态更新
   - 生产进度跟踪
   - 发货安排

3. **订单变更**:
   - 修改订单内容
   - 变更审核流程

### 发货管理

1. **创建发货单**:
   - 基于销售订单创建
   - 选择发货产品和数量
   - 填写收货地址
   - 选择物流方式

2. **发货执行**:
   - 库存出库
   - 生成发货单据
   - 物流跟踪

### 销售退货

1. **退货申请**:
   - 客户提出退货申请
   - 填写退货原因
   - 审核退货申请

2. **退货处理**:
   - 收货检验
   - 库存入库
   - 退款处理

## 常见问题

### 登录问题

**Q: 忘记密码怎么办？**
A: 联系系统管理员重置密码。

**Q: 登录后页面空白？**
A: 检查浏览器兼容性，建议使用Chrome、Firefox等现代浏览器。

### 操作问题

**Q: 数据保存失败？**
A: 检查必填字段是否完整，网络连接是否正常。

**Q: 权限不足？**
A: 联系管理员分配相应权限。

### 性能问题

**Q: 系统响应慢？**
A: 可能是网络问题或数据量过大，联系技术支持。
