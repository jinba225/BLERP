## 🎨 智能页边距功能

系统会根据纸张类型**自动设置最佳页边距**：

| 纸张类型 | 默认页边距 | 说明 |
|---------|----------|------|
| 热敏小票 | 2mm | 节省纸张 |
| 标签 | 1mm | 最大化打印区域 |
| 快递单 | 3mm | 适应打印机 |
| 小尺寸纸张 | 5mm | A6, A7, B6等 |
| 标准纸张 | 10mm | A4, A5, B5, Letter等 |
| 大尺寸纸张 | 15mm | A3, B4, Legal等 |

**智能调整规则：**
```javascript
// 纸张宽度或高度 < 120mm → 页边距 5mm
// 纸张宽度或高度 > 250mm → 页边距 15mm
// 标签类型 → 页边距 1mm
// 热敏小票 → 页边距 2mm
```

## 🔄 横向/纵向切换

**所有纸张都支持横纵向切换：**

```
A4 纵向: 210mm × 297mm (常用)
A4 横向: 297mm × 210mm

58mm小票 纵向: 58mm × 210mm (常用)
58mm小票 横向: 210mm × 58mm (罕见)
```

**注意：**
- 标签和小票通常只用纵向
- A4、Letter等标准纸张两种方向都常用
- 横向适合宽表格、横幅

## 📊 尺寸对照表

### A系列 vs B系列

| A系列 | 尺寸 | B系列 | 尺寸 | 大小关系 |
|------|------|------|------|---------|
| A3 | 297×420 | B4 | 250×353 | B4 < A3 |
| A4 | 210×297 | B5 | 176×250 | B5 < A4 |
| A5 | 148×210 | B6 | 125×176 | B6 < A5 |

**规律：** B系列比同级A系列小，但比下一级A系列大
```
A4 (210×297) > B5 (176×250) > A5 (148×210)
```

### 标准纸张 vs 北美纸张

| ISO标准 | 北美标准 | 差异 |
|---------|---------|------|
| A4 (210×297) | Letter (216×279) | Letter稍宽但短 |
| - | Legal (216×356) | 更长的Letter |

```
1. 选择"自定义尺寸"
2. 输入宽度和高度
3. 手动设置页边距
4. 点击"应用"
```

## 📝 常见问题

### Q1: 我的打印机不支持某个尺寸怎么办？

**A:**
1. 检查打印机设置中的支持列表
2. 使用"自定义尺寸"功能调整
3. 考虑换用标准尺寸（如A4）

### Q2: 标签尺寸可以横向打印吗？

**A:** 可以，但不推荐。标签通常纵向打印。

### Q3: 热敏小票高度可以调整吗？

**A:** 可以。高度210mm只是默认值，实际按内容自动调整。

### Q4: 如何自定义名片尺寸（90×54mm）？

**A:**
```
1. 选择"自定义尺寸"
2. 宽度：90
3. 高度：54
4. 页边距：2mm
5. 点击"应用"
```
